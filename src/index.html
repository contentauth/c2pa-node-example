Sign buffer then save file

This calls route /upload, to attach the manifest to the image and sign with the test certificate, then save the file in the uploaded_images directory. NOTE: Only JPEG or PNG formats are supported.

Upload then sign file

This calls route /upload_file_sign to upload the file and save it. Then it attaches a manifest and signs it with the test certificate, saving that file, too.

Thus, for each file uploaded there are two files saved in the uploaded_images directory: one unsigned and one signed.
