Sign buffer THEN save file

This saves the file after attaching the manifest and signing with the test cert. Thus, there is only one file saved in the uploaded_images directory. Calls route /upload.

Upload THEN sign file

This uploads the file, then attaches a manifest and signs it with the test cert., saving that file, too. Thus, for each file uploaded there are two files saved in the uploaded_images directory: one unsigned and one signed. Calls route /upload_file_sign.
